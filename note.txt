Catppuccin in fcitx5:
	git clone https://github.com/catppuccin/fcitx5.git
mkdir -p ~/.local/share/fcitx5/themes/
cp -r ./fcitx5/src/* ~/.local/share/fcitx5/themes
Subsequent steps: 
    Navigate to Fcitx5 Configuration application through your application launcher.
    Select the Addons Tab.
    Select the setting icon (gear-wheel) for Classical User Interface, located to the right.
    Apply Catppuccin as the desired fcitx5 theme by navigating to Theme and later selecting Catppuccin Flavour.
